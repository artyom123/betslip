# Betslip

Sports betting application.

Betslip use:

* React
* Redux
* React router
* eslint airbnb
* css

## Getting Started

```
npm start
```

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Running the tests

```
npm run test
```
